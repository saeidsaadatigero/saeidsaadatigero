### 🚀 About Me

I am a results-driven **Senior Software Engineer** based in Iran with **3+ years of production experience** architecting scalable web systems and deploying robust AI models. My core philosophy revolves around **Clean Architecture, High Availability, and Workflow Automation**. I specialize in bridging the gap between complex AI/ML research and resilient, production-grade backend infrastructure.

- 🏗️ **Current Focus:** Architecting High-Frequency Trading Bots & Distributed Async Systems.
- ⚡ **Superpower:** Designing secure API Gateways and bypassing complex network topologies.
- 🧠 **Mindset:** "Make it work, make it right, make it fast, then automate it."
